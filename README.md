# Виртуальная физическая лаборатория
## Возможности
В этом проекте пользователь может поставить эксперимент над телефоном, падающим с различных зданий на разных небесных телах.

**Список зданий:**
1. Обычный жилой трехэтажный дом.
2. Биг Бэн.

**Список небесных тел:**
1. Луна
2. Земля
3. Марс
4. Юпитер
